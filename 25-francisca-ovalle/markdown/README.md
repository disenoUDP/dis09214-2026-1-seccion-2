# MACKDOWN
---
## GITHUB: 
Nube diseñadora en donde se puede almacenar codigos, al almacenarlos se crea un red o arbol genealogico lo que te permite ingresa a
codigos que se ocuparon hace mucho tiempo.
Es similar a un word markdown en donde se puede escribir el texto con los distintos formatos y ver el progreso realizado.  
## MACKDOWN:  
Lenguaje de marcado ligero, una forma de escribir el tecto con formatos añadiendo caracteres comunes del teclado.  

---  
#### ¿CÓMO ESCRIBIR EN MACKDOWN?  

**Párrafo nuevo:** Enter x2   
**Bajar una línea:** Espacio x2 + Enter  
**Sangría:** TAB o Espacio x4  
**Forzar salto:** <br*>

#### ENCABEZADOS:  
# TITULO 1  
## TITULO 2  
### TITULO 3
#### TITULO 4


**comnd + ++**: Agrandar letra
#### FORMATO DE TEXTO:  

**Negrita**: * *Negrita**  
*Cursiva*: * Cursiva*  
***Negrita y Cursiva***: *** Texto***  
~Tachado~: ~Tachado ~

#### LISTAS:
1. **Viñetas y números:** (Espacio x2 + Enter y después Espacio x4)
* Elemento A: 
* Elemento B:
2. **Subelementos**: (Espacio x2 + Enter después Espacio x4, ELEMENTO y Espacio x2 + Enter)
3. **Checkbox** : (guión, espacio, corchetes, ELEMENTO, Espacio x2 + Enter)
   - [x] Elemento C
   - [x] Elemento D
#### LINKS Y IMÁGENES:  

**Links**: [Texto](URL)  
**Imagen**: [Texto](URL IMAGEN)  
**Imagen más link**: [![Texto](URL imagen)](URL destino)
