**Diagramas de flujo:** diagramación gráfica de un algoritmo de los pasos de un proceso. en programación, se utiliza como una herramienta de planificación para visualizar la lógica de un programa antes de escribir una sola linea de código
Se usan componentes estándar (simbología) para que cualquier programador pueda entenderlo.

- Ocuparemos Javascript, más simplificado
- setup: su propósito es configurar el entorno inicial. define elmtamaño
- respetar mayúsculas y minusculas!!!
- createCanvas+(ancho,alto): tamaño del canvas
- colores en rgb: https://htmlcolorcodes.com/
- funciona en ejes (X,Y)
- stroke (puntos, lineas y bordes de la figura): le asigna el color
- strokeWeight: asigna grosor
- todo lo que este abajo de un stroke, sigue su configuracion. si queremos que sea diferente hacemos otro stroke 
- noStroke(); : quitar el borde
